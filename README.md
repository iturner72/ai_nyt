# AI NYT

This is a project meant to conquer this bounty set by Bajajis: https://warpcast.com/balajis.eth/0x218b92a7.

## Installation

```bash
git clone https://github.com/iturner72/ai_nyt.git
```

### Client 
```bash
cd client && npm i
```

```bash
npm start
```bash

### Server
```bash
cd server && cargo run
```
